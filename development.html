Development and Compliance

Planning and Design

The website was planned for a local animal rescue organisation called Happy Paws Pet Adoption. The main goal was to create a simple, clear, and user-friendly website for people who want to learn about pet adoption, view available pets, and submit enquiries.

Development Process

The website was developed using HTML and CSS in Visual Studio Code. Each page was created separately and linked through a consistent navigation menu. The design uses readable text, clear headings, appropriate images, and simple page layouts.

Privacy Issue

The Contact Us form collects a user's name, email address, and comments. This information should only be used to respond to the enquiry and should not be shared with other people or organisations. A privacy notice was included on the Contact Us page.

Security Issue

A basic security issue is that online forms can collect personal information. To reduce risks, the form only asks for necessary details and uses required fields and email input validation. No sensitive information such as passwords or payment details is collected.

Copyright Issue

Images used on the website should be copyright-safe. Suitable free image sources such as Unsplash, Pexels, or Pixabay can be used. Image sources should be acknowledged where required.

Censorship and Content Appropriateness

The website should only display appropriate and family-friendly content. Images and text should not be offensive, misleading, or harmful. Since the website is about pet adoption, all content was kept positive, clear, and suitable for general visitors.
